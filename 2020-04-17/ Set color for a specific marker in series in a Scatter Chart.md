###  Set color for a specific marker in series in a Scatter Chart


[Set color for a specific marker in series in a Scatter Chart 路 Issue #1247 路 apexcharts/apexcharts.js](https://github.com/apexcharts/apexcharts.js/issues/1247 "Set color for a specific marker in series in a Scatter Chart 路 Issue #1247 路 apexcharts/apexcharts.js")


 

```js
series: [
  {
    name: 'Points',
    data: [
      {
        x: 1,
        y: 2.14
      },
      {
        x: 3.2,
        y: 5.8
      },
      {
        x: 3.8,
        y: 6.04,
        fillColor: '#111'
      },
      {
        x: 4.55,
        y: 6.77
      }
    ]
  }
]
```
